توضیح ضروری : پیشاپیش ازینکه نمی تونم به سوالاتتون جواب بدم معذرت می خوام برای هر نوع اطلاعاتی بازم گوگل رو پیشنهاد میکنم و کلمه کلیدی هم leiomyosarcoma   هست برای جزئیات می تونین کلماتی مثل amputation ( قطع عضو ) یا surgery ( جراحی ) یا treatment ( درمان ) رو سرچ کنین
